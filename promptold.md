重要度ランキング（理由とざっくり目安つき）

FCF利回り（FCF / 時価総額）
　→ 将来キャッシュの“期待利回り”。利益よりブレにくく、投資妙味を直接見る。
　目安：5%以上で割安感、2%未満は割高気味。

EV/EBITDA
　→ 資本構成（借金の多寡）を調整した“企業全体の倍率”。会計差にも比較的強い。
　目安：5〜10倍がよくある帯。二桁後半は期待先行の可能性。

FCFマージン（FCF/売上）＋営業利益率の“セット”
　→ 「会計上の利益」と「実際の現金」の両輪。乖離が大きければ要注意。
　目安：**営業利益率↑かつFCFマージン↑**が理想。片方だけ高いのは要因を深掘り。

成長の質（売上・営業利益の5年CAGR + CCR）
　→ 伸び続けていて、しかも利益が現金化（CCR≥1）しているか。
　目安：売上/営利とも右肩、CCR≥1が安定。

ネット有利子負債 / EBITDA
　→ 借金を“何年で返せるか”。景気逆風に耐えられるかの核心。
　目安：3倍以下が無難、1倍未満は盤石。

インタレスト・カバレッジ（EBIT/利息）
　→ 金利上昇に対する耐性。
　目安：5倍以上で安心、2倍割れは要警戒。

ROIC / ROCE（資本効率）
　→ 投下資本でどれだけ稼げているか。ROEより歪みにくい“本命”。
　目安：10〜15％超で強い企業が多い。

PEG（PER/成長率）
　→ 成長株の“PERの割高さ”を成長で割り戻す。
　目安：≈1で妥当、<1なら相対的に割安。

PSR / EV/Sales
　→ 赤字〜低利益局面でも比較可能。マージンと必ずセット確認。
　目安：成熟産業は1倍前後、SaaS等は**>5倍**でも妥当なことあり。

配当性向・配当成長（3年CAGR）
　→ 還元の持続性と成長性。無理配当や希薄化の見抜きに。
　目安：性向**30〜60%**が心地よい帯、CAGRは業種次第。

当座比率 / 現金比率（短期安全）
　→ 目先の資金繰り。流動比率より厳しめで見れる。
　目安：当座100％超、現金比率は高いほど余裕。

PBR
　→ 簿価基準の割安度。資産型（金融・不動産・商社など）では有効、グロースには効きにくい。
　目安：1倍割れは“解散価値＜時価”の示唆だが、理由の吟味が必須。

価格トレンド（3年の傾き：Up/Flat/Down）
　→ ファンダが良くても“落ちるナイフ”は避けたい時のタイミング用フィルター。
　使い方：基本はDownを避ける。少なくともFlat以上で入ると心理的にも楽。

どう使い分ける？（超ショートカット）

まず割安さ：①FCF利回り → ②EV/EBITDA

次に実力の裏取り：③FCFマージン＋営業利益率、④成長CAGR＋CCR

倒れにくさ：⑤ネットD/EBITDA、⑥ICR

仕上げ：⑦ROIC、⑧PEG、⑨PSR/EV/Sales、⑩配当、⑪短期安全、⑫PBR

最後にタイミング：⑬トレンドがDownなら見送り（改善待ち）

“買いゾーン”の目安（暫定ルール例）

FCF利回り ≥ 5% かつ EV/EBITDA ≤ 10

売上・営業利益が右肩、CCR ≥ 1

ネットD/EBITDA ≤ 3、ICR ≥ 5

トレンド：Flat以上
→ これを満たすなら、業種中央値との比較で違和感がないかを最後にチェック。

例外：金融/REIT/資源などは勘定構造が違うので、PBR・NAV・在庫/市況など専用指標を優先。
